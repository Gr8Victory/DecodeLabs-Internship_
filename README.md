# DecodeLabs-Internship_
Week 3 – Project 3 SQL Data Analysis 

## Overview
This project analyzes 1,200 e-commerce transactions using Microsoft SQL Server to uncover business insights, evaluate sales performance, and identify growth opportunities.

## Tools Used
- Microsoft SQL Server
- SQL (T-SQL)

## Key Highlights
- Total Revenue: $1.26M+
- Total Orders: 1,200
- Total Units Sold: 3,535
- Average Order Value: $1,053.97

## Analysis Performed
- Dataset Exploration
- KPI Analysis
- Product Performance Analysis
- Revenue by Payment Method
- Referral Channel Analysis
- Monthly Sales Trend Analysis

## Key Findings
- Chairs and Printers were the top-performing products.
- Instagram generated the highest number of orders.
- June recorded the highest revenue and order volume.
- High cancellation and return rates indicate operational challenges.
